## 问题修复：记忆检索 user_id 不匹配

### 问题现象

用户在 `/new` 命令后重新对话，询问 AI "你的名字叫什么"，AI 无法从记忆中找到用户之前给起的名字，但 `/memory list` 命令可以正常显示该记忆。

### 根因分析

**问题 1: user_id 不匹配**

| 操作 | user_id | 来源 |
|------|---------|------|
| 存储记忆 (`archive_with_reme`) | '张烽林' | USER.md 解析 |
| 检索记忆 (`RetrieveMemoryTool`) | 'default_user' | 默认值（未传递） |
| 列出记忆 (`ListMemoriesTool`) | 'default_user' | 默认值（未传递） |

**根因**：`RetrieveMemoryTool` 和 `ListMemoriesTool` 的 `execute()` 方法未使用正确的 user_id，默认使用 `'default_user'`，导致无法检索到存储在 `'张烽林'` 用户下的记忆。

**问题 2: ReMe 内部模型能力不足**

日志显示：
```
ValueError: retrieve_memory: missing required input 'query_items'
```

ReMe 的 `PersonalRetriever` agent 使用 ReAct 循环调用内部工具，需要模型生成 `query_items` 参数。**LongCat-Flash-Lite 模型的工具调用能力弱**，无法正确生成该参数。

同时发现 ReMe 内部仍使用 `'default_user'`：
```
'memory_target': 'default_user'
```

说明 ReMe 内部 `user_name` 到 `memory_target` 的参数映射存在问题。

### 修复方案

#### 修复 1: 工具传递正确的 user_id

**修改文件**：`nanobot/agent/tools/memory.py`

**修改内容**：

```python
# RetrieveMemoryTool - 已在上一轮修复
class RetrieveMemoryTool(Tool):
    def __init__(self, adapter: RemeMemoryAdapter, get_user_name: Callable[[], str] = _default_get_user_name):
        self._adapter = adapter
        self._get_user_name = get_user_name
    
    async def execute(self, query: str, top_k: int = 5, **kwargs: Any) -> str:
        user_id = self._get_user_name()
        result = await self._adapter.retrieve_memory(query, user_id=user_id, top_k=top_k)

# ListMemoriesTool - 本次修复
class ListMemoriesTool(Tool):
    def __init__(self, adapter: RemeMemoryAdapter, get_user_name: Callable[[], str] = _default_get_user_name):
        self._adapter = adapter
        self._get_user_name = get_user_name  # 新增
    
    async def execute(self, limit: int = 10, **kwargs: Any) -> str:
        user_id = self._get_user_name()  # 使用正确的 user_id
        memories = await self._adapter.list_memories(user_id=user_id, limit=limit)

# register_memory_tools() - 本次修复
def register_memory_tools(registry, adapter, get_user_name):
    registry.register(RetrieveMemoryTool(adapter, get_user_name))
    registry.register(AddMemoryTool(adapter, get_user_name))
    registry.register(ListMemoriesTool(adapter, get_user_name))  # 新增传递
    registry.register(GetMemoryStatusTool(adapter))
    registry.register(DeleteMemoryTool(adapter))
```
